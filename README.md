# Учебная Виртуальная Машина (УВМ) - Вариант 28

## Описание проекта

Реализация ассемблера и интерпретатора для учебной виртуальной машины (УВМ) согласно варианту 28 задания. Проект включает полный набор инструментов для работы с УВМ: ассемблер, интерпретатор, графический интерфейс и тестовую программу.

## Структура файлов проекта

| Файл | Назначение |
|------|------------|
| `uvm_asm_v28.py` | Ассемблер - преобразует текстовые команды в бинарный код |
| `uvm_interp_v28.py` | Интерпретатор - выполняет бинарный код и сохраняет результаты |
| `simple_ui.py` | Графический интерфейс (Tkinter) для удобной работы |
| `test_program_v28.csv` | Пример тестовой программы |

## Быстрый старт

### 1. Клонирование/создание проекта

Создайте папку для проекта и поместите в нее все файлы:

- `uvm_asm_v28.py`
- `uvm_interp_v28.py`
- `simple_ui.py`
- `test_program_v28.csv`

### 2. Проверка работы ассемблера и интерпретатора

```bash
# Ассемблирование тестовой программы
python uvm_asm_v28.py -i test_program_v28.csv -o program.bin -t 0

# Выполнение программы
python uvm_interp_v28.py -i program.bin -o result.json -r '0-150'

# Просмотр результатов
python -c "import json; print(json.dumps(json.load(open('result.json')), indent=2))"
